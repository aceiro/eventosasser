Bem-vindo ao sistema de submissão de resumos. Aqui, você vai encontrar os passos para submeter (enviar) o seu trabalho. Leia os passos com atenção e siga os itens indicados.


  • Modelos
  • Enviar Resumo
  • Verificar Status
  • Re-enviar Resumo

Aqui você econtra o primeiro passo para enviar o seu resumo para avaliação. Para facilitar a edição do seu trabalho use os arquivos de exemplos disponíveis aqui. Nesse caso, o arquivo do Ms/World contém o formato do resumo do evento e o arquivo do Ms/PowerPoint descreve a formatação para o do painel para o dia da apresentação.

ATENÇÂO: para enviar o resumo, você vai precisar editar apenas o arquivo .doc. O arquivo .ppt deve ser utilizado apenas para o dia da apresentação.

Aqui você encontra o segundo passo para enviar o seu resumo para avaliação, logo após você ter editado o seu resumo no Ms/World.

ATENÇÂO: Verifique com o seu orientador e/ou professor responsável se o seu resumo está pronto para ser enviado. Caso o resumo não atenda os requisitos de formatação do evento o resumo deverá ser re-enviado pelo sistema.

Caso seu resumo esteja pronto para ser enviado, clique no ícone abaixo.

Esse é o terceiro passo para o envio do resumo para o evento. Aqui você poderá conferir o status se está aprovado ou se precisa ser re-enviado. Para verificar o seu resumo, clique no ícone abaixo para continuar.

Finalmente, nesse passo você poderá re-enviar o seu resumo. Para isso, clique no ícone para continuar.

ATENÇÂO: Faça o re-envio do seu resumo apenas se o mesmo não foi aceito.
