Bem-vindo ao sistema de submissão de resumos. Aqui, você vai encontrar os passos para submeter (enviar) o seu trabalho. Leia os passos com atenção e siga os itens indicados.


  • Passo 1 - Modelos
  • Passo 2 - Enviar Resumo
  • Passo 3 - Status do Resumo

Aqui você encontra o primeiro passo para enviar o seu resumo para avaliação.

Para facilitar a edição do seu trabalho use os arquivos de exemplos disponíveis. no formato Ms/World e Ms/PowerPoint

ATENÇÃO: para enviar o resumo, é preciso editar apenas o arquivo .doc. O arquivo .ppt deve ser utilizado apenas para o dia da apresentação.

Após você ter editado o seu resumo no Ms/World, copie e cole as partes importantes do seu trabalho nos campos indicados no formulário de submissão.

Para acessar o formulário de submissão, faça o cadastro acessando o ícone abaixo.

ATENÇÃO: Verifique com o seu orientador, professor ou coordenador se o seu resumo está pronto para ser enviado. Caso o resumo não atenda os requisitos de formatação do evento o resumo deverá ser re-enviado pelo sistema.

Caso seu resumo esteja pronto para ser enviado, clique no ícone abaixo.
